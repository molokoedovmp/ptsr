СТРУКТУРА КУРСА:
━━━━━━━━━━━━━━━━
Курс (Course)
  └─ Модуль 1 (CourseModule)
       ├─ Урок 1 (Lesson)
       ├─ Урок 2 (Lesson)
       └─ Урок 3 (Lesson)
  └─ Модуль 2 (CourseModule)
       ├─ Урок 1 (Lesson)
       └─ Урок 2 (Lesson)


СПОСОБ 1: ЧЕРЕЗ СКРИПТ
━━━━━━━━━━━━━━━━━━━━━━

1. Откройте файл scripts/add-lessons-to-module.js

2. Измените эти переменные:
   const COURSE_SLUG = 'slug-kursa'  // slug курса из админки
   const MODULE_ORDER = 0             // номер модуля (0, 1, 2...)

3. Отредактируйте массив lessons:
   const lessons = [
     {
       title: 'Название урока',
       description: 'Краткое описание',
       content: 'Полное содержание урока в Markdown',
       videoUrl: null,  // или 'https://youtube.com/...'
       duration: 15,     // длительность в минутах
       isFree: true,     // бесплатный урок?
       orderIndex: 0,    // порядок урока
     },
     // ... другие уроки
   ]

4. Запустите скрипт:
   npx ts-node scripts/add-lessons-to-module.js


СПОСОБ 2: ЧЕРЕЗ АДМИНКУ (РЕКОМЕНДУЕТСЯ)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Зайдите в админку: http://localhost:3000/admin/courses

2. Найдите нужный курс и нажмите "Редактировать" (иконка карандаша)

3. Прокрутите до секции "Модули курса"

4. Каждый модуль можно:
   - Редактировать название и описание
   - Изменить порядок
   - Удалить

5. В будущем здесь появится возможность добавлять уроки прямо из админки!


КАК НАЙТИ SLUG КУРСА:
━━━━━━━━━━━━━━━━━━━━━
1. Откройте админку курсов
2. Slug отображается под названием курса
3. Или посмотрите в URL курса: /programs/[ЭТО-SLUG]


КАК УЗНАТЬ НОМЕР МОДУЛЯ:
━━━━━━━━━━━━━━━━━━━━━━━
Модули нумеруются с 0:
- Первый модуль = 0
- Второй модуль = 1
- Третий модуль = 2
И т.д.


ФОРМАТ КОНТЕНТА УРОКА:
━━━━━━━━━━━━━━━━━━━━━━
Используйте Markdown для форматирования:

## Заголовок 2
### Заголовок 3

**Жирный текст**
*Курсивный текст*

- Список
- Элементов

1. Нумерованный
2. Список

> Цитата или важная информация

```
Код или примеры
```
